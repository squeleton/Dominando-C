# Dominando C

Este repositorio sirve como punto de partida para aquellos estudiantes que quieran tener una guía simplificada y unificada para aprender C.

Además de esto, con el objetivo de comprender realmente el funcionamiento de la programación en C, se enseñarán algunos conceptos generales de computación.

## Ambientes y herramientas

1. Ingresar a este VS Code online con tu cuenta de GitHub: https://cs50.dev/ (Esto porque tiene muchas herramientas preinstaladas, de manera que puedes empezar a aprender C sin preocuparte por ahora de más).

2. Enrolarte en CS50 2025 (opcional)

3. ¡Nada más! Esto lo hago para que el inicio sea lo más amigable posible. Luego meteremos las manos en muchas cosas, no se preocupen.
